# TMS9981Aビデオカード

## 概要
* PC-8001などからTMS9981Aを制御してビデオ出力を行うための拡張基板

## 使用環境
* PC-8001+PC-8012
* PC-8001mk2/SR
* PC-8801シリーズ（未確認）

## I/Oポート
|I/O|アドレス|用途|
|:--|:--|:--|
|IN/OUT|$98|データポート|
|IN/OUT|$99|コントロールポート|
